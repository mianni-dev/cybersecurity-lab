# Cybersecurity Lab

This repository documents my hands-on cybersecurity labs performed in a virtual lab environment using Kali Linux and vulnerable machines.

The goal of this project is to build practical security skills in:

- Network reconnaissance
- Vulnerability assessment
- Web application security
- Password attacks
- Exploitation and privilege escalation
- Security hardening and defense

## Lab Environment

Virtualization: VirtualBox  
Attacker Machine: Kali Linux  

Additional vulnerable systems will be added as labs progress.
